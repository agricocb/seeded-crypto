Detailed Description

A PrivateKey is used to unseal messages sealed with its corresponding PublicKey. The PrivateKey and PublicKey are generated from a seed and a set of derivation specified options in JSON Format for Derivation Options.

The PrivateKey includes a copy of the PublicKey, which can be reconstituted as a PublicKey object via the getPublicKey method.
