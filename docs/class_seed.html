Detailed Description

A seed/secret, which is itself (re)derived from another secret seed and set of key-derivation specified options in Key-Derivation Options JSON Format.

Because seed derivation uses a one-way function, this seed can be shared without revealing the secret used to derive it. It can then be used and, if lost, re-derived from the original seed and keyDerivationOptionsJson that were first used to derive it.
