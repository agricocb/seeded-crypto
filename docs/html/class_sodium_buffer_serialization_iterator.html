Detailed Description

An iterator for popping fields out of a message constructed with SodiumBuffer.pushField.

Example:

SodiumBuffer field1(...);
SodiumBuffer field2(...);
// Making field1 an expicit field in the message, with (superflous) length bytes
// in the internal representation
SodiumBuffer message = SodiumBuffer().pushField(field1).pushField(field2);
SodiumBufferSerializationIterator i = SodiumBuffer.getSerializationIterator();
SodiumBuffer poppedField2 = i.popField();
SodiumBuffer poppedField1 = i.popField();
// Making field1 an implciit field in the message, which is retrieved via
// popRemainder
SodiumBuffer message = field1.pushField(field2);
SodiumBufferSerializationIterator i = SodiumBuffer.getSerializationIterator();
SodiumBuffer poppedField2 = i.popField();
SodiumBuffer poppedField1 = i.popRemainder();

◆ popField()

const SodiumBuffer SodiumBufferSerializationIterator::popField ( )

Pop a field that was previously pushed onto a SodiumBuffer message stack. Fields are returned in first-in-last-out (FILO) order.

If the creation of the original SodiumBuffer started with a call to SodiumBuffer::pushField on an empty sodiumBuffer, popField should be used to pop the final field. If the first field Sodium buffer constructed from data that was never pushed, the final field is returned with popRemainder.

Returns
const SodiumBuffer
